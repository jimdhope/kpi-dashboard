# **App Name**: KpiQuest

## Core Features:

- KPI Tracking: Capture and store KPIs across campaigns, pods, and teams with user roles and permissions.
- Achievement Visualization: Visualize individual and team achievements with leaderboards and progress bars.
- AI-Powered Motivation: Generate personalized encouragement messages based on user progress toward their KPIs. The tool will use reasoning to decide what information to incorporate to the message.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of progress and achievement.
- Secondary color: Light gray (#F0F0F0) for backgrounds and neutral elements.
- Accent: Gold (#FFD700) to highlight achievements and milestones.
- Clean and modern sans-serif fonts for readability and a professional feel.
- Use clear and consistent icons to represent different KPIs and achievements.
- Dashboard layout with clear sections for individual, team, and pod progress.
- Subtle animations to celebrate achievements and provide visual feedback.
