# Daily Word dataset

The bundled Daily Word lists were derived from
[`steve-kasica/wordle-words`](https://github.com/steve-kasica/wordle-words),
revision `3288e72bd4d305a2388f34c3620a930612365f50`.

- `daily-word-answers.json` contains the 2,315 words whose CSV `day` field is
  populated.
- `daily-word-allowed.json` contains all 12,972 unique five-letter words in the
  source CSV, including every answer.

The repository was archived on 2 July 2024. The data is bundled at build time;
KPI Quest does not make runtime requests to GitHub.
